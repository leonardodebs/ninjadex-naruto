# Changelog - Ninjadex

Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

## [2.0.0] - 2026-03-14

### ✨ Novas Funcionalidades
- **Ninjadex Core:** Transformação completa do exemplo base em uma enciclopédia ninja funcional.
- **Sistema de Busca:** Implementação de busca em tempo real por nome de Shinobi.
- **Filtros Avançados:** Adição de filtros por Vila Oculta, Rank Ninja, Raridade e Dōjutsu.
- **Modo Comparação (Sharingan):** Interface dedicada para comparar estatísticas de dois ninjas lado a lado com gráficos de radar.
- **Temas Dinâmicos:** Suporte completo para Modo Escuro (Dark Mode) e Modo Claro, com persistência visual.
- **Easter Egg:** Adição do "Modo Chidori" ativado por sequência de teclas.

### 🛡️ Segurança e Performance
- **Análise de Segurança:** Auditoria completa de requisitos de segurança (XSS, Referrer Policy, Gestão de Segredos).
- **Referrer Policy:** Implementação de `no-referrer` em todas as mídias externas para maior privacidade.
- **Otimização de Imagens:** Integração com proxy de imagens para carregamento mais estável de assets da Wikia.
- **Lazy Loading:** Implementação de carregamento preguiçoso para imagens na galeria principal.

### 📝 Documentação
- **README.md:** Criação de documentação completa em português com guia de tecnologias e estrutura.
- **Metadata:** Atualização de metadados oficiais do app.

*Nota: Este release marca a transição de um protótipo para uma aplicação funcional completa.*
